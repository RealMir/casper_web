포렌식이란?

  디지털 데이터를 근거로 삼아 해당 디지털 기기에서 발생한 사실을 알기 위해 조사 분석하는 것
  조사분석에는 데이터 복구, 암호복호화(패스워드 크랙),데이터 검색 등이 있다.

(1) 데이터 복구

  • 데이터가 삭제되었을 경우 복구한다.

  • 기본적인 파일 삭제는 메타정보와 데이터의 링크만
    사라져서 원본데이터는 살아있어 복구가 가능하다.
 - 관련 툴 : HexEditor
  • 16진수 값(Hex)을 편집할 수 있음

  • 파일 시그니처 확인 및 편집

  • 해시값 확인

(2) 데이터 검색

  • 사용자가 은닉한 정보를 식별한다.

  • 데이터 은닉 기법 : 원본 파일을 다른 형식의 파일안에
    숨기는 기법인 스테가노 그래피 기법이 있다.
 - 스테가노 그래피 ( 실습 화면 )
의심되는 .gif형식의 사진을 확보
 HxD 돌려본 결과 => xss코드를 가져 위험

(3) 활용 방향

경찰이 범죄에서 디지털 증거를 획득하기 위해 사용
대표적 예시) 최순실 태블릿, 승리 / 정준영 카톡
