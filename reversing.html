리버싱이란?

  프로그램을 분석하여 구조와 기능, 적용 기술을 역으로 따라가며 분석하고 재현하는 것
  분석방법에는 정적분석과 동적분석이 있다.

(1) 정적 분석

  • 파일을 실행시키지 않고 겉모습을 관찰.

  • 파일의 종류, 크기, 헤더, 등록정보 등의
    다양한 내용 확인.
 - 관련 툴 : HexEditor
  • 16진수 값(Hex)을 편집할 수 있음

  • 파일 시그니처 확인 및 편집

  • 해시값 확인

(2) 동적 분석

  • 정적분석을 토대로 분석한다.

  • 파일을 직접 실행해 파일, 레지스트리, 네트워크 등을
    관찰하여 프로그램의 행위를 분석한다
 - 관련 툴 : OllyDbg
  • 바이너리 코드를 분석을 위한 디버거
    소스코드 없이 실행파일만으로 분석

  • 어셈블리 코드와 레지스터 상태,
    스택등을 확인 할 수 있다.

(3) 활용 방향

포렌식과 같이 범죄현장 분석에 사용되고 있다.
트로이 목마, 웜 바이러스, 스파이웨어 같은 악성코드 분석에도 사용한다.
