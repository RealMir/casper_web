네트워크 해킹이란?

  네트워크 상에서 발생되는 해킹을 의미한다.
  크게 Sniffing, Spoofing, DDoS공격 세가지 절차 로 나눌 수 있다.

(1) 스니핑 ( Sniffing )

  • '킁킁 거리다'의 의미로 네트워크 흐름의 중간에서
    도청 및 감시하는 공격 행위

  • 패스워드 등의 데이터를 통신하는 과정에서 중간에
    도청하여 확인 할 수 있어서 위험함
 - 관련 툴 : Nmap
  • 해킹 대상의 시스템의 "정보 수집"

  • 어떤 서비스(ex : telnet, ssh, ftp등)가
    작동 되고 있는지 수집

  • 어떤 포트들이 열려있는지 내용을
    수집

(2) 스푸핑 ( Spoofing )

  • '변조시키다'의 의미로 사용자가 원하는 행위를 속여
    비정상적인 행동을 하는 행위

  • 공격 예) DNS 스푸핑, ARP 스푸핑

 - 관련 툴 : Wire Shark
패킷 캡처 => 로그인 암호 확인

(3) DDOS

엄청난 양의 패킷을 보내 네트워크를 과부하 시켜
해당 서비스를 이용자들이 사용할 수 없게 만드는 공격
