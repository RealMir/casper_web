웹 해킹이란?

  웹 사이트의 취약점을 공격하는 기술적 위협으로 웹 페이지를 통해 공격한다.
  SQL Injection, XSS, CSRF, 웹 셸 업로드 기법등이 있다.

(1) SQL Injection

  • 데이터베이스와 연동되어 있는 애플리케이션의
    입력값을 조작하는 공격 기법이다

  • SQL 쿼리문에 기반하여 공격을 수행해 인증 우회,
    DB 데이터 조작 및 유출할 수 있다
 - 실습 화면
로그인 창에 쿼리문 입력
 쿼리문에 의해 로그인 우회됨

(2) XSS

  • 클라이언트에 대한 취약점을 이용한 javascript와
    HTML 언어를 사용한 공격법이다

  • 악의적인 코드를 사용자의 웹 브라우저에 심어놓으면
    접속시 악성코드가 실행된다
 - 실습 화면
로그인 창에 코드 입력
 로그인 창 넘어가면서 코드 실행됨

(3) 관련 툴

Burpsuite : DB의 칼럼을 확인해 SQL Injection에 활용한다

Flddler : 파라미터 변조 및 패킷을 확인 할 수 있다
