포너블이란?

  시스템 해킹을 말하며, 시스템의 일반 유저 자격으로 여러 가지 공격 기법을 활용해 관리자 자격을 뺏는 절차
  버퍼오버플로우, 익스플로잇 공격기법이 있다.

(1) 버퍼오퍼플로우(BOF)

  • 메모리에 할당된 버퍼의 양을 초과하는 데이터를
    입력하여 프로그램의 주소를 조작한다.

  • 사용자 명령어 라인의 해석기 Shell로 제어를 넘겨서
    셀코드를 이용해 프로그램에 접근한다.
 - 실습 화면
쉘코드를 이용해 BOF를 발생시킴

(2) 익스플로윗

  • 취약점 공격으로 공격 대상 컴퓨터의 제어 권한 획득

  • 컴퓨터의 소프트웨어나 하드웨어의 버그,
    보안 취약점 등 설계상 결함을 이용해 공격하는 기법
 - 관련 툴 : Web Exploit ToolKit
  • 웹 사이트를 이용하여 악성코드
    유포를 쉽게 수행

  • 공격 페이지 난독화 및 지능적인
    은닉화가 가능하다

(3) 더 알아보기

다른 해킹과 다르게 정리가 잘 된 서적이 없어서 공부할 때 불편하므로
어셈블리어, c언어, 쉘 코드등 컴퓨터 언어에
의존해야해서 많은 지식이 요구된다.
